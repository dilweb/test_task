# restfulapi

Самый простой способ запуска — через Docker Compose.  
Установите Docker и Docker Compose, затем выполните: `docker compose up --build`  
Откройте: `http://localhost:8000` (доки API обычно: `http://localhost:8000/docs`)  
Остановка: `docker compose down`  

Для локальной разработки нужен Python 3.14.x и Poetry.  
Установите Poetry по официальной инструкции: https://python-poetry.org/docs/#installation  
Далее: `poetry install`
